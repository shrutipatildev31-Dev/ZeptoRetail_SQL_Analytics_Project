Zepto Retail Analytics Dashboard

Project Overview

This project focuses on analyzing Zepto retail data using SQL and Power BI to generate business insights related to revenue, inventory management, product discounts, and pricing strategies.

The project includes data cleaning, SQL analysis, DAX calculations, KPI generation, and interactive dashboard visualizations.

---

Tools & Technologies Used

- PostgreSQL
- SQL
- Power BI
- DAX
- CSV Dataset

---

Project Workflow

1. Imported raw retail dataset into PostgreSQL
2. Performed data cleaning and preprocessing using SQL
3. Analyzed product, inventory, and discount data
4. Created business insight queries using SQL aggregations and CASE statements
5. Imported cleaned dataset into Power BI
6. Created calculated columns and DAX measures
7. Built interactive dashboard visualizations and KPI cards

---

Data Cleaning Performed

- Checked NULL values
- Removed products with zero MRP
- Converted paise values into rupees
- Validated product categories
- Checked duplicate product names

---

SQL Analysis Performed

- Revenue analysis by category
- Top discounted products analysis
- Inventory stock analysis
- Price per gram analysis
- Weight category distribution
- Average discount analysis
- Inventory weight calculation

---

DAX Calculations Used

- Total Revenue
- Total Products
- Average Discount
- Total Categories
- Inventory Weight
- Price Per Gram
- Weight Category classification

Dashboard Features

- Discount analysis
- Product distribution insights
